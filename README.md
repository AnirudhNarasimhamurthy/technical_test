# Submission for the technical test

Instructions to run the program:

The main file is izi.py

To run the program ensure MySQLdb package is installed in the system and the system has Python2.7.
To install MySQLdb in Mac/Linux use the following command :
pip install MySQL-python

Once that is done, execute the code using the following command:

python izi.py

And the text interface will hopefully lead you through the application
